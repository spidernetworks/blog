üì¶ Installing Docker

Below are the most common installation commands:

üê≥ Running Your First Container

This downloads an image from Docker Hub and runs it inside a container.

üìÇ Listing Images and Containers

# List all containers including stopped

üõ† Creating Your First Dockerfile

Example: a simple Python web app.

# Dockerfile
FROM python:3.11-slim

WORKDIR /app

COPY requirements.txt .
RUN pip install -r requirements.txt

COPY . .

CMD ["python", "app.py"]

Build your image:

docker build -t my-python-app .

Run it:

docker run -p 5000:5000 my-python-app

Now visit http://localhost:5000.

üöÄ Deploying a Docker App to a Server

Basic deployment steps:

  3. Pull your image
